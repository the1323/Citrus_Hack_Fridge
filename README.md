# Inside_Fridge
## How to run:
### FLASK_APP = app.py
### FLASK_ENV = development
### FLASK_DEBUG = 0
### cd your_folder
### /Users/your_folder/venv/bin/python -m flask run
### App will be Running on http://127.0.0.1:5000
